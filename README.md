# 우아한테크코스 7기 미션 PR History

우아한테크코스에서 총 8개의 미션을 수행하며 JavaScript·React 생태계 전반을 실전 과제 중심으로 학습했습니다. 각 미션은 1단계(페어 프로그래밍)와 2단계(리팩토링)로 구성되어, 협업을 통해 문제 해결 과정을 고도화하고 이후 개선 작업으로 코드 품질을 단계적으로 끌어올렸습니다. 또한 모든 미션을 현업자 리뷰와 피드백으로 반복 검증하며, 구현을 넘어 설계·품질 관점까지 확장할 수 있었습니다.

초반 4개 미션(자동차 경주, 로또 등)에서는 바닐라 JavaScript 기반의 핵심 개념과 구현 역량을 단단히 다졌고, 후반 4개 미션(페이먼츠, 모듈 등)에서는 React 기반 SPA 개발을 바탕으로 상태 관리, 컴포넌트 설계, 모듈화 등 프론트엔드 아키텍처 역량을 강화했습니다.
